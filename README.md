# Q-xal Academy — Landing Page
**Q-xal Academy** es el sitio web oficial (Landing Page) del proyecto educativo Q-xal, diseñado para presentar información, servicios, cursos y contacto de forma clara, moderna y atractiva.

Este repositorio contiene exclusivamente el **frontend público** del proyecto, construido sobre **Laravel** para ofrecer un entorno flexible, escalable y con un desarrollo limpio.

---

## Tecnologías Principales

- **Laravel 10+** — Framework base del proyecto  
- **Blade** — Motor de plantillas  
- **Bootstrap 5 / Tailwind (si aplica)** — Estilos y componentes  
- **JavaScript / Alpine.js (si aplica)** — Interactividad  
- **MySQL / PostgreSQL / SQLite** — (Opcional según el entorno)

---

## Objetivo del Proyecto

El propósito de esta landing page es:

- Presentar la identidad oficial de **Q-xal Academy**  
- Ofrecer información clara sobre cursos, talleres y servicios  
- Recibir registros o formularios desde el público  
- Proveer una interfaz moderna, rápida y responsiva  
- Servir como entrada al ecosistema educativo de Q-xal
